# Projeto CRUD com Spring Boot, JPA e Maven

Sistema desenvolvido com o objetivo de aplicar os principais conceitos de **desenvolvimento backend em Java**, implementando um **CRUD completo**, **tratamento de exceções** e seguindo o **padrão em camadas** (Controller, Service, DTO, Entity, Repository, Exception e Handler).  
O projeto utiliza o **Spring Framework** e boas práticas de arquitetura para garantir código limpo, escalável e de fácil manutenção.

---

## Tecnologias Utilizadas

- **Java 21** - Versão LTS (Long-Term Support), moderna e com melhorias de performance.
- **Spring Boot** - Framework para criação de aplicações Java de forma rápida e padronizada.
- **Spring Data JPA** - Simplifica o acesso e manipulação de dados no banco relacional.
- **Hibernate** - Implementação do JPA, responsável pelo mapeamento objeto-relacional.
- **Maven** - Gerenciador de dependências e automação de build.
- **H2** - Banco de dados em memória utilizado para testes e desenvolvimento local.
- **Jakarta Validation** - Para validações automáticas de campos com anotações.
- **Spring Web** - Fornece suporte à criação de APIs RESTful e controle de requisições HTTP.
